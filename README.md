## 信用卡付款失败？先排查这几个问题

付款失败是搬瓦工买家最常见的卡点。原因通常就这几种：

**1. 卡片不支持境外消费**  
国内银行卡默认可能关闭了境外消费权限。打开银行 App 或拨打客服热线，确认"境外消费"和"网络支付"功能已开通。

**2. 卡信息填写有误**  
卡号、有效期、CVV 任何一项填错都会直接拒绝。账单地址（Billing Address）也很重要，要和你在银行备案的地址一致，不一致的话 Stripe 风控会拦截。

**3. 信用额度不足**  
确认卡内余额或可用信用额度高于账单金额。

**4. 被风控系统拦截**  
部分银行对境外 VPS 类交易比较敏感，直接拦截。这时候换支付方式是最简单的解决办法：切换到支付宝或银联，一般就能顺利完成。

**5. 开了代理导致 IP 和账单地址不匹配**  
购买时如果开着代理，IP 归属地和你填的国家（China）对不上，系统会判定为欺诈。关掉代理用本地 IP 重新下单即可。

## 常见问题 FAQ

**Q：国内双币信用卡直接能用吗？还是需要开通什么权限？**

A：双币卡（Visa 或 Mastercard）通常可以直接使用，但部分银行默认关闭境外网络消费功能，建议付款前先在银行 App 确认"境外消费"和"网络交易"权限已开启。工行、建行、招行、交行的双币卡普遍反馈可用。

**Q：搬瓦工信用卡付款会自动续费吗？可以取消吗？**

A：绑定信用卡后，系统默认在到期时发起续费扣款。可以在后台进入对应服务，手动关闭自动续费选项，关闭后到期不会自动扣款，需要手动续费。

**Q：付款失败，订单还在吗？**

A：在。搬瓦工的订单在付款失败后一般保留 24 小时，你可以登录账户在 "Billing - My Invoices" 里找到未付款的账单，换个支付方式（比如支付宝）重新付款就行，不需要重新下单。

**Q：搬瓦工支持用虚拟信用卡付款吗？**

A：部分虚拟卡可以，但成功率不稳定，尤其是一些临时虚拟卡会被 Stripe 的风控系统拦截。建议优先使用实体信用卡或支付宝，更稳。

**Q：信用卡付款用美元结算，汇率怎么算？**

A：搬瓦工所有套餐定价为美元，信用卡扣款时按付款当日银行汇率换算成人民币，由发卡银行处理，换算汇率和手续费参考各家银行的规定。如果你的卡有境外消费返现或积分，用信用卡反而更划算。
